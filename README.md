## üèÅ Getting Started

### Prerequisites
-   Node.js 20+
-   pnpm 9+

### Installation

```bash
git clone https://github.com/maicol07/pcgw_editor.git
cd pcgw_editor
pnpm install
```

### Development

```bash
pnpm run dev
```

### Production Build

```bash
pnpm run build
```
